# Developer Portfolio

A premium, award-winning developer portfolio built with Next.js 15, React 19, Framer Motion, GSAP, Three.js, and Tailwind CSS.

## Features

- Cinematic loading screen with progress counter
- Smooth scrolling with Lenis + GSAP ScrollTrigger
- Custom cursor with trail and glow effects
- 3D floating object (React Three Fiber)
- Glassmorphism UI with aurora gradients
- Magnetic buttons and scroll reveals
- Command palette (Ctrl+K)
- Dark/Light theme toggle
- Project filtering and search
- Animated testimonials carousel
- Contact form with validation
- Konami code easter egg
- Spotify widget & GitHub activity graph
- Fully responsive and accessible

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **UI:** React 19, Tailwind CSS, Shadcn UI
- **Animation:** Framer Motion, GSAP, Lenis
- **3D:** React Three Fiber, Drei
- **Icons:** Lucide React, React Icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customization

Edit `lib/data.ts` to update your personal information, projects, skills, and experience.

Update `siteConfig` with your name, email, and social links.

Replace the profile image URL in `sections/about.tsx`.

Add your resume PDF to `public/resume.pdf`.

## Build

```bash
npm run build
npm start
```
